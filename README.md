## Getting Started

To get a copy of this project up and running locally for development and testing purposes, fork the repo, then:

```
git clone https://github.com/YOUR_USERNAME/ubiquitous-spoon.git 
cd ubiquitous-spoon
npm install  
npm run dev
```
